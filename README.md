## 🚀 Функциональные возможности

### Основной функционал

- ✅ **Просмотр списка партнеров** с отображением:
  - Тип / Наименование партнера
  - ФИО директора
  - Контактный телефон
  - Рейтинг
  - Автоматически рассчитанная скидка

- ✅ **Управление партнерами**:
  - Добавление нового партнера
  - Редактирование существующего
  - Удаление партнера (с каскадным удалением продаж)

- ✅ **Просмотр истории продаж** выбранного партнера с колонками:
  - Наименование продукции
  - Количество
  - Дата продажи
  - Сумма (в рублях)

- ✅ **Управление продажами**:
  - Добавление новой продажи
  - Редактирование продажи
  - Удаление продажи
  - Автоматический расчет общей суммы

- ✅ **Автоматический расчет скидки** на основе объема продаж:
  - до 10 000 шт. → 0%
  - 10 000 – 50 000 шт. → 5%
  - 50 000 – 300 000 шт. → 10%
  - более 300 000 шт. → 15%

### Дополнительные возможности

- ✅ **Экспорт данных** в форматы TXT, CSV, JSON
- ✅ **Импорт данных** из JSON
- ✅ **Контекстное меню** (ПКМ) для быстрого доступа к операциям
- ✅ **Валидация вводимых данных** с информативными сообщениями
- ✅ **Подтверждение критических действий** (удаление, выход)
- ✅ **Обработка исключений** с понятными сообщениями пользователю
- ✅ **Ограничение на ввод** (максимальное количество, максимальная сумма)

## 🎨 Интерфейс приложения

### Главное окно

Главное окно разделено на две области:
- **Левая панель** – список партнеров
- **Правая панель** – история продаж выбранного партнера
- **Верхнее меню** – доступ ко всем функциям

### Окно добавления/редактирования партнера

Содержит поля:
- Наименование (обязательное)
- Тип партнера (выпадающий список)
- Рейтинг (только цифры)
- Адрес
- ФИО директора
- Телефон
- Email

### Окно добавления/редактирования продажи

Содержит:
- Выбор продукции из списка
- Количество (с проверкой лимита)
- Дата продажи
- Автоматически рассчитанная сумма

## ⚙️ Конфигурация

### Строка подключения к базе данных

В файле `CherkasovLibrary/AppDbContext.cs`:

```csharp
protected override void OnConfiguring(DbContextOptionsBuilder optionsBuilder)
{
    optionsBuilder.UseNpgsql("Host=localhost;Database=cherkasov;Username=app;Password=123456789;Encoding=UTF8");
}
```

### Настройка схемы базы данных

По умолчанию используется схема `app`. При необходимости можно изменить в моделях:

```csharp
[Table("partners_cherkasov", Schema = "app")]
```
